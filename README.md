## preliquisites
1. php and composer
2. mysql server
3. node

## Project setup
1. clone project
2. run `npm install`
3. run `npm run watch` to compile sass into css
4. setup database and create a copy of `.env.sample` name it `.env` 
5. run `php artisan serve`
